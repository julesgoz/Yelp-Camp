# Yelp-Camp
This is a Yelp clone with full CRUD capabilities.  

Users can:
  - Sign-up
  - Login
  - Create new campgrounds
  - Edit/delete campgrounds
  - Upload pictures
  - Write reviews

Hosted Version:  
<a href="https://serene-depths-72342.herokuapp.com/">Link</a>

Built with:
- JS + ejs
- MongoDB
- NodeJS

Based on Colt Steel's Web Dev BootCamp
